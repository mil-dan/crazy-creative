# Сайт компании **Crazy Creative**, занимающейся дизайном.

Одностраничный адаптивный лендинг, разработанный в рамках учебного пет-проекта для отработки навыков верстки и JavaScript.

Посмотреть, как выглядит сайт, можно по ссылке:

## 🛠 Технологии
*   **HTML5**
*   **CSS3 / SCSS (Sass)**
*   **JavaScript (ES6)**
*   **Методология БЭМ**

## ✨ Реализованный функционал

*   **Адаптивная верстка:** Корректное отображение на экранах любой ширины.
*   **Бургер-меню:**
    *   Открытие/закрытие меню по клику на бургер-кнопку.
    *   Блокировка скролла страницы при открытом меню.
    *   Закрытие меню по клику на затемненный фон.
*   **Переключатель контента в секции Hero (смена изображения):**
    *   При клике на элемент `#switch` меняется состояние индикатора.
    *   Изображение `#switchImg` меняется с `hero-3.svg` на `hero-4.svg` и обратно.
    *   Добавлен визуальный эффект плавного угасания для мягкой смены картинок.
*   **Аккордеон (секция FAQ):**
    *   При клике на вопрос происходит раскрытие ответа.
    *   **Умное поведение:** при открытии нового вопроса предыдущий автоматически закрывается.
*   **Кнопка «Наверх»:**
    *   Кнопка появляется при скролле вниз (более 20px от верха страницы).
    *   Реализована плавная прокрутка в начало страницы (`behavior: "smooth"`).

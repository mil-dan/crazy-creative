## ✨ Реализованный функционал

*   **Адаптивная верстка:**
*   **Бургер-меню:** Реализация навигации для мобильных устройств на чистом JS (открытие/закрытие по клику).
*   **Смена контента:** Кнопка в секции `hero__switch` динамически меняет изображение.
*   **Аккордеон:** Блок `faq` с раскрытием и закрытием текста при клике на заголовок вопроса.
